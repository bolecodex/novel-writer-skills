# 情绪价值量化模型

## 核心公式

```
虐点冲击力(M-Impact) = 事件伤害度 × (主角初始期望 - 主角当前韧性)
仇恨值累积(H-Value) = M-Impact × 反派残忍度
爽点回报率(S-Payoff) = 反转事件强度 × 累积H值 × 延迟满足系数
情绪价值(E-Value) = -10(极致虐心) 到 +10(极致爽快)
```

## 10章节奏控制

| 章节 | E-Value范围 | 阶段 | 要求 |
|------|------------|------|------|
| 第1章 | 0 → -3 | 虐点建立 | 不要出现回忆虐点，用当下冲突 |
| 第2章 | -3 → -5 | 虐点递进 | 反派压迫升级 |
| 第3章 | -5 → -7 | 虐点加深 | 希望破灭感 |

| 第6章 | -5 → -2 | 反击初步 | 小试牛刀 |

## 期待感营造技巧

### 悬念设置
- 问题链：每解决一个问题产生三个新问题
- 后果预告：暗示严重后果但不明说

### 情绪钩子
- 共鸣点设置：选择读者最能代入的场景
- 恐惧放大：将普通恐惧极端化
- 欲望刺激：报复、正义、爱情的满足预期

### 节奏控制
- 快慢交替：紧张情节后必有缓冲
- 高潮预热：在大高潮前设置小高潮
- 情绪留白：让读者自己脑补最痛苦的部分

## 第4章付费点卡点技巧

### 1. "生死关头"式卡点
在主角遭遇致命威胁，行动结果悬而未决的关键瞬间切断。

### 2. "惊天逆转"式卡点
在某个颠覆读者认知的重大秘密即将被揭晓的临界点切断。

### 3. "关键抉择"式卡点
将主角置于极其艰难、影响深远的两难困境面前，在即将做出决定的瞬间切断。

### 4. "情感爆发"式卡点
在人物间的情感冲突达到最高潮、情绪最饱满的瞬间切断。

### 5. "奖励触手可及"式卡点
目标就在眼前，但获取它还需要克服最后一道突发障碍时切断。

## 爆款公式模板

### 公式A：重生复仇打脸文
```
1-2章：重生觉醒，发现前世真相
3-4章：伪装配合，暗中收集证据
5-6章：初步反击，小试牛刀
7-8章：证据曝光，真相大白
9-10章：全面清算，圆满复仇
```

```

7-8章：疯狂挽回，屡屡碰壁
9-10章：彻底拒绝，各自结局
```

```
7-8章：男主追悔，拒绝复合

```

## 创新虐点元素库

1. 科技虐点：朋友圈屏蔽、转账记录、定位欺骗

3. 医疗虐点：器官捐赠、试管婴儿、亲子鉴定
