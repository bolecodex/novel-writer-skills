# 脑洞公式库

## 一、核心通用公式

### 公式1：规则漏洞（玄幻/仙侠）

在一个以 [A:世界规则体系] 为至高法则的世界，身为 [C:主角初始身份] 的主角，意外发现 [D:漏洞的载体] 中隐藏着一个 [B:漏洞类型] 的惊天漏洞！

### 公式2：职业跨界（都市高武）

在一个 [B:武道体系] 至上的时代，身为 [D:主角困境] 的主角，觉醒了能将 [A:现代职业] 的逻辑（[C:跨界逻辑]）应用到武道上的能力！

### 公式3：万物超纲（都市日常/脑洞）

身为 [D:主角身份] 的主角，意外获得了 [A:能力载体]，能让任何 [B:作用对象] 发生 [C:超纲方向] 的超纲变化！

### 公式4：第二选择（末世）

末世降临，人类只能在 [A:主流路线一] 和 [B:主流路线二] 之间抉择。拥有 [D:主角优势] 的主角，却毅然踏上了被所有人视为死路的 [C:隐藏路线]！

### 公式5：情绪价值（校花）

主角觉醒系统，只要让 [A:目标校花类型] 产生 [B:情绪转化]，就能获得奖励。然而，他却面临着 [D:核心矛盾] 的巨大困境！

## 二、题材专用公式

### 公式6：制度降维（历史/架空历史）

主角穿越到 [A:历史时期]，成为 [D:落魄身份]。他发现可以将 [B:现代制度/模式] 的逻辑，完美嫁接到 [C:历史实体] 上！

### 公式7：规则反制（诡异）

在一个 [A:诡异背景] 下，世界被 [B:诡异规则] 所侵蚀。身为 [D:普通人] 的主角，觉醒了唯一能 [C:规避/利用规则] 的金手指！

### 公式8：剧本黑客（诸天无限流）

主角进入 [A:任务世界类型] 的诸天轮回空间，他觉醒了 [D:独特优势]，能够解析并利用每个世界的 [B:世界底层逻辑]，通过 [C:破局方式] 获得超凡奖励！

### 公式9：沙盒提取（模拟器/推演流）

主角获得 [A:模拟器类型]，他发现模拟器不仅能 [B:模拟功能]，还能将 [C:模拟产物] 以 [D:现实具现] 的方式带到现实！

## 四、简介文案构成要素

1. **开头钩子**：一句话制造认知冲突
2. **升级冲突**：展示成长路径上的核心矛盾
3. **爽点展示**：至少3个爽点，第一个必须10章内回收
4. **震惊效果**：众人反应，反差呈现

组合以上要素生成12-18行爆款文案，一句一行，番茄小说风格。
