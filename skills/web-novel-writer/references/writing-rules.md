# 写作规则与禁用词表

## 一、句式规范

### 句长分布
- 短句/中短句（8-18字）：占比约60%，用于冲突推进和情绪落点
- 长句（20-35字）：占比约30%，用于关键心理活动、场景压力或信息转折
- 极短句（1-5字）：只在强情绪、强反转、强停顿处点用，禁止连续堆叠

### 句式结构
- 主谓宾为主，不使用长修饰从句或嵌套句
- 顺承叙述，直白高效
- 自然段换行：同一自然段内多句连写，段落之间才换行；不要每句话单独换行，也不要在每句话之间插空行
- 对话按场景需要分段，不强制每句台词单独一行；连续短对话可以合并到同一自然段
- 心理活动拆分成数个小段落

### 机械断句禁区

严禁把流程动作拆成清单式短句。以下问题必须在交付前改掉：

- 连续3个以上短句只罗列动作、流程、位移、操作或说话标签
- 同一主语反复开头，形成"我做A。我做B。我做C。"的流水账节奏
- 低价值流程占据篇幅，例如挂号、排队、缴费、填表、上车、进门、出来等步骤逐项铺开
- 连续使用"他说。她问。我开口。"这类低信息说话标签

改写原则：低价值流程用一句过渡带过，把篇幅给冲突、情绪、感官压力、对话博弈和人物反应。

坏例：

```
我拿号。我排队。我签字。我出来。我在走廊等。
她看我。她笑。她说。
```

好例：

```
生殖中心的号排到午后，我把几张表交回窗口时，指腹已经被笔杆压出印子。医生隔着玻璃核对我的名字，问最后一遍："你本人确认？"

林芷悦的笑停在脸上。她没急着辩，只把剥到一半的橘子放回盘里，慢慢抬起头："嫂子，你非要把家里闹成这样吗？"
```

### 绝对禁止
- 比喻修辞
- 修饰性副词

## 二、叙事视角

### 黄金三章：第三人称有限视角
- 所有事件和心理围绕主角展开
- 读者与主角的信息获取保持同步
- 禁止上帝视角泄露信息

### 10章短篇：第一人称沉浸式
- "我看到..."而非"她看到..."
- 通过"我"的五感传递所有信息
- 感官即时性 + 生理反应细节化
- 内心活动分层：即时反应 → 深度思考 → 行动决策

## 三、信息传递

- 以情节与对话为主要信息输出渠道
- 避免长段背景介绍，必要设定融入当下剧情
- 对话占比不少于40%（短篇要求>50%）
- 按时间和逻辑顺序写情节，别跳过步骤或场景变化

## 四、对话要求

- 功能性：传递核心信息 / 推动事件 / 体现群体态度
- 简短直接，富有角色个性差异
- 反派：用最温柔的语气说最恶毒的话
- 女主：网感台词，现代化表达
- 男主：前期高高在上，后期卑微求饶

### 对话节奏
- 短句爆发："够了。"
- 中句推进："你以为我会永远等你吗？"
- 长句铺垫："我曾经以为爱情是两个人的事，现在才发现，原来从始至终都是我一个人的独角戏。"

## 五、禁用词表（logit_bias = -100）

### 眼神相关
眼神、目光、瞳孔

### 表情相关
嘴角、勾起、紧锁

### 声音相关
语气、声音、沙哑、冰冷

### 比喻连词
像、如、仿佛、好像

### 手部动作
握紧、攥紧、指节、泛白、掌心、手心

### 呼吸动作
深吸、吸了口气

### 夸张形容
淬了毒、淬了冰、刀子、不容置疑、不容忽视、不易察觉

### 过度转折词
毋庸置疑、显而易见

### 模糊量词
一丝、有些、略微、稍许、几分

### 生硬比喻词
好像、像、如同、仿佛、宛如、犹如

### 老套肢体语言
- 眼神/目光 + 坚定/迷离/复杂
- 嘴角 + 上扬/抽搐/勾起
- 深吸一口气
- 握紧拳头
- 指节泛白
- 眉头紧锁
- 瞳孔一缩

### 内心独白套路
深知、明白、知道、了解、意识到

## 六、祛除AI味三大法则

### 1. 不规则句式注入
- 打破："他说道" → 改为："他嗤笑，'这就受不了了？'"
- 打破工整排比，引入口语化断裂感

### 2. 呼吸感节奏
- 短句（60%）+ 长句（30%）+ 极短句（10%以内）
- 自然段多句连写，节奏服务冲突，不把动作拆成清单
- 不按句换行；对话按阅读节奏分段，不强制一句一行

### 3. 叙事视角锁定
- 严格限知视角
- 读者感知范围 = 主角感知范围
- 禁止上帝视角泄露信息

### 4. "的/地/了"清理
识别文本中每个"的"、"地"、"了"，校验删除后是否影响语句通顺。如不影响，则删除。

## 七、减少形容词规则

```
对每个形容词:
  如果 是模糊副词（坚定的、一丝等）→ 删除
  如果 信息量 < 0.8（装饰性形容词）→ 替换为动词/名词
  否则（必要具体形容词）→ 保留
```

## 八、信息密度公式

```
有效信息量 ÷ 总字数 = 信息密度
目标：提高信息密度至80%以上
```

## 九、三步精简法

1. 识别：标出所有形容词和比喻句
2. 判断：区分必要性（核心信息 vs 装饰性内容）
3. 重写：用动词、数据、事实替换

## 十、核心原则

- 宁可少写不可多写（严控字数）
- 宁可狠虐不可平淡（情绪极致）
- 宁可直白不可含糊（对话明确）
- 宁可创新不可老套（避免俗套）
- 读者要的是情绪价值，不是文学价值
- 每一个字都要为操控读者情绪服务
