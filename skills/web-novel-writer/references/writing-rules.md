# 写作规则与禁用词表

## 一、句式规范

### 句长分布
- 短句/中短句（8-15字）：占比70%，主体节奏
- 长句（20-35字）：占比30%，仅用于关键心理活动或能力效果
- 极短句（1-5字）：频繁点缀，制造停顿

### 句式结构
- 主谓宾为主，不使用长修饰从句或嵌套句
- 顺承叙述，直白高效
- 单句成段：大量使用只有一句话的段落
- 对话独立分段
- 心理活动拆分成数个小段落

### 绝对禁止
- 比喻修辞
- 修饰性副词

## 二、叙事视角

### 黄金三章：第三人称有限视角
- 所有事件和心理围绕主角展开
- 读者与主角的信息获取保持同步
- 禁止上帝视角泄露信息

### 10章短篇：第一人称沉浸式
- "我看到..."而非"她看到..."
- 通过"我"的五感传递所有信息
- 感官即时性 + 生理反应细节化
- 内心活动分层：即时反应 → 深度思考 → 行动决策

## 三、信息传递

- 以情节与对话为主要信息输出渠道
- 避免长段背景介绍，必要设定融入当下剧情
- 对话占比不少于40%（短篇要求>50%）
- 按时间和逻辑顺序写情节，别跳过步骤或场景变化

## 四、对话要求

- 功能性：传递核心信息 / 推动事件 / 体现群体态度
- 简短直接，富有角色个性差异
- 反派：用最温柔的语气说最恶毒的话
- 女主：网感台词，现代化表达
- 男主：前期高高在上，后期卑微求饶

### 对话节奏
- 短句爆发："够了。"
- 中句推进："你以为我会永远等你吗？"
- 长句铺垫："我曾经以为爱情是两个人的事，现在才发现，原来从始至终都是我一个人的独角戏。"

## 五、禁用词表（logit_bias = -100）

### 眼神相关
眼神、目光、瞳孔

### 表情相关
嘴角、勾起、紧锁

### 声音相关
语气、声音、沙哑、冰冷

### 比喻连词
像、如、仿佛、好像

### 手部动作
握紧、攥紧、指节、泛白、掌心、手心

### 呼吸动作
深吸、吸了口气

### 夸张形容
淬了毒、淬了冰、刀子、不容置疑、不容忽视、不易察觉

### 过度转折词
毋庸置疑、显而易见

### 模糊量词
一丝、有些、略微、稍许、几分

### 生硬比喻词
好像、像、如同、仿佛、宛如、犹如

### 老套肢体语言
- 眼神/目光 + 坚定/迷离/复杂
- 嘴角 + 上扬/抽搐/勾起
- 深吸一口气
- 握紧拳头
- 指节泛白
- 眉头紧锁
- 瞳孔一缩

### 内心独白套路
深知、明白、知道、了解、意识到

## 六、祛除AI味三大法则

### 1. 不规则句式注入
- 打破："他说道" → 改为："他嗤笑，'这就受不了了？'"
- 打破工整排比，引入口语化断裂感

### 2. 呼吸感节奏
- 短句（80%）+ 长句（20%）+ 极短句点缀
- 大量使用只有一句话的段落
- 对话独立分段

### 3. 叙事视角锁定
- 严格限知视角
- 读者感知范围 = 主角感知范围
- 禁止上帝视角泄露信息

### 4. "的/地/了"清理
识别文本中每个"的"、"地"、"了"，校验删除后是否影响语句通顺。如不影响，则删除。

## 七、减少形容词规则

```
对每个形容词:
  如果 是模糊副词（坚定的、一丝等）→ 删除
  如果 信息量 < 0.8（装饰性形容词）→ 替换为动词/名词
  否则（必要具体形容词）→ 保留
```

## 八、信息密度公式

```
有效信息量 ÷ 总字数 = 信息密度
目标：提高信息密度至80%以上
```

## 九、三步精简法

1. 识别：标出所有形容词和比喻句
2. 判断：区分必要性（核心信息 vs 装饰性内容）
3. 重写：用动词、数据、事实替换

## 十、核心原则

- 宁可少写不可多写（严控字数）
- 宁可狠虐不可平淡（情绪极致）
- 宁可直白不可含糊（对话明确）
- 宁可创新不可老套（避免俗套）
- 读者要的是情绪价值，不是文学价值
- 每一个字都要为操控读者情绪服务
