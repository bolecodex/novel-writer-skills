# 10章短篇阶段

写一部完整的10章短篇小说。总字数：9000-12000字。

## 输入

- `{导语}`：用户提供的开篇概要（200-400字，分10行）
- `{情节设计}`：用户的情节设计笔记
- 题材方向和故事公式（如果来自脑洞阶段）

## 写前准备：加载参考资料

1. 读取 [../references/writing-rules.md](../references/writing-rules.md) 获取风格规则和禁用词。
2. 读取 [../references/emotion-model.md](../references/emotion-model.md) 获取情绪价值模型。
3. 如果是女频题材，还需读取 [../references/villain-dialogue-examples.md](../references/villain-dialogue-examples.md) 获取反派台词模式。

## 执行：串行规划 → 并行写作 → 串行缝合

### 阶段A：串行 — 大纲与情绪曲线设计

在写任何正文之前，先设计完整架构：

1. **选择故事公式**：
   - 公式A：重生复仇打脸文
   - 公式B：追妻火葬场
   - 公式C：替身虐恋翻身
   - 或根据用户概念设计自定义公式

2. **设计10章大纲**，每章包含：
   - 一句话情节概要
   - 目标情绪值（E值：-10到+10）
   - 章末钩子/悬念
   - 具体的反派台词时刻

3. **设计情绪曲线**：
   - 第1-4章：虐点层层递进，E值从0降到-8以下
   - 第4章（付费点）：情绪最低谷，最强悬念钩子
   - 第5-7章：开始反转，E值缓慢回升
   - 第8-10章：爽点爆发，E值达到+8以上

4. **设计导语**（如果用户没提供）：
   - 200-400字，分10行，每行20-40字
   - 第一行：陈述句，快速建立核心人物关系和背景
   - 第二行：现状陈述，交代即将发生的关键事件
   - 第三行：引入矛盾对话（绿茶女配/偏心男主）
   - 第四行：矛盾升级，推向第一个情绪高潮
   - 第五至十行：主角的决绝反应，信息差钩子

将大纲呈现给用户，获得批准后再继续。

### 阶段B：并行 — 分组写作

并行启动 **3个独立任务**，每个写一组章节。将完整大纲和情绪曲线传递给每个任务。

**任务1：第1-3章（虐点建立）**
- E值：0 → -5 → -7
- 建立主角的苦难、反派的残忍
- 第1章：最大化震撼/愤怒感，不要用回忆虐点
- 对话驱动（>50%），反派台词必须恶毒

**任务2：第4-6章（付费点+反转）**
- E值：-8 → -6 → -3
- 第4章必须使用付费点卡点技巧：
  - 生死关头式（致命危机悬而未决）
  - 惊天逆转式（大秘密即将揭晓）
  - 关键抉择式（两难困境）
  - 情感爆发式（冲突达到顶点）
  - 奖励触手可及式（最后一道障碍）
- 第5-6章：主角开始反击

**任务3：第7-10章（爽点爆发）**
- E值：0 → +4 → +7 → +9
- 全面反转，主角碾压
- 第10章：故事收官，圆满结局
- 累积仇恨值（H-Value）全部转化为爽点回报

每个任务必须：
- 每章严格1000-1200字
- 使用第一人称沉浸式视角
- 对话占比>50%
- 第1-9章结尾必须有悬念钩子（不要用内心独白式威胁）
- 遵循 references/writing-rules.md 中的所有写作规则

### 阶段C：串行 — 缝合与一致性检查

3个任务全部返回后：

1. **衔接审查**：检查章节边界是否自然过渡，修正角色姓名不一致、时间线断裂、语气偏移。
2. **连贯性检查**：验证前期埋下的伏笔在后期是否得到回收。
3. **CLI 校验**：

```bash
novel-cli validate <输出文件>
```

4. 修复发现的任何问题（字数违规、禁用词）。

## 输出格式

```
【导语】
（200-400字，分10行）

【第1章】
（正文，每句一行，1000-1200字）
[本章字数：XXXX]

【第2章】
...

【第10章】
（正文，每句一行，1000-1200字）
[本章字数：XXXX]

【总字数】：XXXXX字
```

## 章节结尾规则（第1-9章）

- 风格：含蓄、冷静、充满张力
- 制造信息差钩子，让读者对下一章产生强烈好奇
- 允许的技巧：意想不到的动作/事件、模棱两可的高信息量对话、关键物品特写、主角不知道的震惊侧面信息

**禁止的结尾模式：**
- 直接对角色喊话（"傅谨言，你..."）
- 空洞威胁词："结束/开始"、"万劫不复"、"一无所有"
- 复仇宣言："享受吧"、"慢慢玩"
- 剧透具体的报复手段
- 咆哮体，缺乏精妙/隐晦
