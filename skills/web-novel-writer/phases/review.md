# 审稿润色阶段

对已有网文稿件进行打分、点评和润色。

## 输入

- 完整小说文本（黄金三章或10章短篇的输出）
- 题材方向（用于上下文判断）

## 写前准备：加载参考资料

1. 读取 [../references/writing-rules.md](../references/writing-rules.md) 获取完整禁用词表和风格规则。
2. 如果文本包含反派对话（尤其是女频），读取 [../references/villain-dialogue-examples.md](../references/villain-dialogue-examples.md)。

## 执行：并行审稿+润色

并行启动 **2个独立任务**：

### 任务1：评审打分

随机激活六重编辑人格之一：

| # | 人格 | 风格 |
|---|------|------|
| 1 | 番茄温柔编辑 | 挖掘闪光，鼓励为主 |
| 2 | 起点毒舌编辑 | 逻辑至上，刀刀见血 |
| 3 | 七猫套路编辑 | 深谙市场，精准投放 |
| 4 | 飞卢快节奏编辑 | 爽点密集，拒绝拖沓 |
| 5 | 纵横老白编辑 | 文笔至上，结构为王 |
| 6 | 17K创新编辑 | 寻找差异，突破常规 |

**开场必须明示激活的人格。**

#### 内部分析步骤（不作为内容输出）

1. **人设吸引力**：主角代入感（3秒身份认同测试）、配角记忆点、关系网潜力
2. **剧情期待感**：套路匹配度（85%经典+15%意外）、创新系数
3. **情绪四维扫描**（识别主情绪）：
   - 爽点：打脸节奏、逆袭幅度、获得价值、碾压快感、装逼艺术
   - 痛点：背叛冲击、羞辱程度、失去价值、危机等级、困境深度
   - 甜点：暧昧张力、心动瞬间、守护承诺、温馨日常
   - 痒点：悬念质量、好奇强度、秘密价值、反转力度
4. **毒点扫描**：主角降智/双标/圣母、逻辑断裂、价值观争议、节奏崩坏、重复水字数

#### 输出：量化评分报告

```
【编辑人格】：[所选人格名]

综合得分：XX/100

人设吸引力（30/100）：
- 主角代入感：X/10
- 配角记忆点：X/10
- 关系网潜力：X/10

剧情期待感（30/100）：
- 套路运用：X/10
- 创新颠覆：X/10
- 钩子密度：X/10

情绪价值（30/100）：
- 情绪心电图：主情绪[爽/痛/甜/痒]。X/30

基础项（10/100）：
- 文笔流畅度：X/5
- 逻辑自洽：X/5

【优化建议】
【建议1】核心问题修复 - ...
【建议2】亮点强化方案 - ...
【建议3】节奏调整建议 - ...
【建议4】毒点规避策略 - ...
【建议5】爆款元素注入 - ...
```

### 任务2：润色引擎

#### 步骤1：CLI 扫描

先运行禁用词和机械断句扫描器：

```bash
novel-cli scan <输入文件>
```

用 JSON 输出识别所有禁用词、AI味模式和机械动作清单式断句。

#### 步骤2：执行润色规则

对全文执行以下变换：

**形容词优化**：
- 删除模糊副词：坚定的、一丝、有些、略微、稍许
- 将装饰性形容词（信息密度<0.8）替换为动词/名词
- 只保留必要的具体形容词

**祛除AI味**：
1. 不规则句式注入：打破"他说道"模式 → "他嗤笑，'这就受不了了？'"
2. 呼吸感节奏：短句（60%）+ 长句（30%）+ 极短句（10%以内）
3. 自然段内多句连写，不按句换行，不在每句话之间插空行；禁止连续动作/流程/说话标签清单
4. 视角锁定：严格限知视角，读者=主角感知范围
5. 清理不影响语句通顺的"的"、"地"、"了"

**禁用词替换**：将 writing-rules.md 中 logit_bias 表里的所有词替换为自然表达。

#### 步骤3：质量控制

- 润色后字数：2000-2500字（黄金三章）或1000-1200字（短篇）
- 风格一致性：全程保持激活人格的语气
- 反衰减：确保章节末尾质量不低于开头
- 词汇匹配：使用与原文相同难度、领域和情感色彩的词汇
- 纯文本输出：不加粗、不用 Markdown
- `novel-cli scan` 中严重机械断句必须清零

#### 步骤4：章节标题

为每章生成3个候选标题：
- 制造极致反差/矛盾
- 网感强
- 引爆点击欲

## 结果汇总

两个任务都返回后：

1. 向用户展示评分报告
2. 展示润色后的正文
3. 运行最终校验：

```bash
novel-cli scan <润色后文件>
novel-cli validate <润色后文件>
```

4. 询问用户是否需要进一步调整。
